# 수랏간 김상궁 랜딩페이지

전통 수제육포 전문점 **수랏간 김상궁**의 정적 HTML 랜딩페이지입니다. 네이버 스마트스토어, 쿠팡, 카카오 채널 CTA와 수제육포 단품/선물세트 안내를 포함합니다.

## 파일 구조

```text
.
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── main-title.png
│   ├── hero-jerky-gift.png
│   └── hero-previous.png
├── README.md
└── .gitignore
```

## 로컬 확인

별도 빌드 과정 없이 `index.html`을 브라우저에서 열면 됩니다.

## GitHub 업로드

1. 이 폴더 전체를 GitHub 저장소에 업로드합니다.
2. `index.html`이 저장소 최상위에 있는지 확인합니다.
3. `assets`, `style.css`, `script.js`도 함께 업로드합니다.

## Vercel 배포

1. Vercel에서 **Add New Project**를 선택합니다.
2. GitHub 저장소를 연결합니다.
3. Framework Preset은 **Other** 또는 기본 정적 사이트 설정을 사용합니다.
4. Build Command는 비워두고, Output Directory도 비워둔 상태로 배포합니다.
5. 배포 후 생성된 URL에서 랜딩페이지를 확인합니다.
